## Usage

Clone into the repository and run using `cargo run --release`.

## Controls

|Button|Function     |
|------|-------------|
|Tab|Mouse lock|
|Arrows, Enter |Control menu|
|WASD|Move|
|Mouse|Look around|
|LMB|Shoot|
|R|Reload|
|Left Alt|Open editor|
|F1|Toggle noclip|
|M|Toggle map (in editor)|
|Escape|Exit|
